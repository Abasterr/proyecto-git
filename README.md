# Proyecto Git

Proyecto de práctica usando Git y GitHub.
Se busca simular un flujo de trabajo colaborativo y demostrar la capacidad de utilizar repositorio, ramas, resolver fusiones y coordinar cambios mediante la generación de Pull Requests

## Funcionalidades
- Página HTML básica
- Barra de navegación (navbar) (Utilizado desde Bootstrap)
- Footer (Utilizado desde Bootstrap)

## Ramas utilizadas
- main: Rama principal
- feature/navbar: Rama creada para el desarrollo del navbar, ejecución manual de conflictos a través de GIT
- feature/footer: Rama creada para el desarrollo del footer, simulación de un flujo remoto a través de la gcreación de una Pool Request desde la web de GitHub
